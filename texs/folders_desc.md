## Описание папок данного проекта(из корня):
(это форк форка Deeppeptide)

.aim/ - хз, артефакт репы от которой форкали, не использовалась
AFToolkit/ - папка с репой для преобразования AF embeddings в per residue vector
baselines/ - также из исходной репы, не использовалась
checkpoints/ - папка для сохранения чекпоинтов моделей
data/ - папка для хранения предподсчитанных эмбеддингов и исходных данных, там есть embeddings_esm2/ - по названию понятно, такая же папка есть
в data/uniprot_2022/embeddings/ - там кроме неё ещё много предподсчитанных эмбеддингов от разных моделей
Там же в data/ лежит файл с айди белков homo sapiens, более подробно про этот и оставшиеся файлы можно понять из файлов запуска, о них далее

в целом папка data/uniprot_2022 - исходный датасет с соответствующей предобработкой, кажется из оригинального deeppeptide, 
при этом есть data/uniprot_2026 - попытка скачать новую ревизию датасета с uniprot, успешно, чуть больше данных, но предобработку не повторили полностью, 
без обработки фланкирующих мотивов как в исходной статье

также в data/ есть несколько папок эмбеддингов, совмещённых с aho-prior, подробнее про него можно почитать в соседних файлах с этим, в папке brainstorm

ещё в корне есть env, думаю понятно что там

evaluation/ - аналогично из форка, не юзали

notebooks/ - часть старая, тоже особо не юзали, но там есть попытка анализа данных, возможно пригодится

old_runs/ - это я переименовал runs/ которая была в исходной форк репе, не юзалось

predictor/ - аналогично из форка

prepare_data/ - аналогично из форка

runs/ - все проведённые эксперименты, в каждой папке есть метрики, логи, и чекпоинт лучшей модели по валидации, также конфиг запуска, и вспомогательный файлик draw_graphics.py с говорящим названием

src/ - папка с кодом, там есть всё для запуска модели, вообще в целом модели запускаются из консоли через run.py, с передачей всех аргументов, и оно уже перенаправляется в src/...
В нём же можно заметить для некоторых файлов много версий, типа train_loop_crf(v1/v2), это всё старые модификации, оставленные на всякий случай, самая новая при этом без указания версии

если погулять по импортам run.py можно понять как вообще устроено решение

texs/ - всякие документы, которые помогают понять что происходит, тут же есть папка overleaf/ - там лежит промежуточный отчёт по проекту, и текущие результаты экспериментов.
Также в texs/Detection_of_proteolytic_peptides_in_protein_sequences.pdf некоторые заметки предыдущих людей, которые работали над форком deeppeptide (напомню, у меня форк форка)
также в texs/ оригинальная deeppeptide статья лежит, лучше её прочитать

в texs/brainstorm/ - некоторые репорты по экспериментам из общения с chatgpt (сомнительного качества идеи, надо перепроверять), возможно пригодится чтобы понять что происходило вообще

также в корне лежит infer.py, для подсчёта большего количества метрик на тесте. 
и ещё infer_with_error_stats.py - некую статистику по ошибкам считает, но файл завайбкодили, тоже перепроверить бы, вообще хз что в нём происходит.
